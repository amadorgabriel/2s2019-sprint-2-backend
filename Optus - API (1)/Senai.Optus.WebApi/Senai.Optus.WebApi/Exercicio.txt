Criar um endpoint para realizar a busca de artistas através do id do estilo.
GET /api/estilos/{id}/artistas

Criar um endpoint para realizar a busca de artistas através do nome do estilo.
GET /api/estilos/buscar/{nome}/artistas

Criar um endpoint para listar a quantidade de estilos disponíveis.
GET /api/estilos/quantidade

Criar um endpoint para listar a quantidade de estilos e a quantidade de artistas disponíveis.
GET /api/admin/dashboard

-- Auten/Verifc.

Incluir a autenticação na aplicação a fim de que somente ADMINISTRADORES: (estilos musicais)

- Cadastrar [ADM]
- Atualizar [ADM]
- Deletar [ADM]

- A visibilidade para listar os artistas é pública. Não precisa de nenhuma restrição.

Porém, qualquer usuário, contanto que esteja autenticado:

- Listar estilos [COMUM]
- Buscar por id. [COMUM]
