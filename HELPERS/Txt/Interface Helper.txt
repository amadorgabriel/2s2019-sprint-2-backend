Então, INTERFACE é o seguinte:	- Uma Camada de Segurança, pois do controler o fluxo 
			 	  fica: Controller - Interface - Repositories
				- Basicamente, o objetivo dela é mostrar o que seu
				  repositorio pode fazer.
COMO FUNCIONA?: 
	- Primeiramente, cara Cria uma class Interface
	- No exemplo: "Quero Listar as Categorias sem Authorizes"
	- Terá um Controller GET /localhost:5000/api/categorias que instanciará 
	  a Interface
	- Nela(interface) terá:
		-> Line: List<Categorias> listarCategorias = new List<Categorias>();
	- Daí no Repositoy:
		-> Lines:
		public namespace CategoriasReposytory : ICategoriasInterface{

			public List<Categorias> ListarAParadaToda(){
				return ctx.Categorias.ToList();
			}
		}

	- É SÓ ISSO MESMO CARA, MAIS SIMPLES IMPOSSIVEL.
